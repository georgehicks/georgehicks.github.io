⚓ Battleship2
Classic 10×10 fleet combat. Two phones, one WiFi, no server. Pair by QR — pure peer-to-peer.
How it works
  1. Both phones on the same WiFi or hotspot (AP isolation off).
  2. Host shows a QR; Joiner scans it, then shows a reply QR.
  3. Host scans the reply — DataChannel opens over the LAN.
  4. Place ships, take turns firing. Hits keep your turn.

iPhone tip: Chrome/Safari block the camera on plain http://192.168.… URLs. Use the always-visible paste code panels (host copies text → joiner pastes → joiner copies reply → host pastes). Camera works on HTTPS or localhost only.
